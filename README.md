### web-app

# Description
this project is designed to help users run their simple standard web apps that uses Node.js and Python easier

## Installation instructions

# Clone the repository
```bash
git clone https://github.com/ahmedhassan-sudo/web-app.git
```
# Navigate into the project folder
```bash
cd web-app
```
# Install dependencies
```bash
cd frontend
npm install
cd ..
```
# Activating the developing enviroment
```bash
cd backend
venv\Scripts\activate
python app.py
```
# Start the development server
```bash
cd frontend
npm start
```

# Usage Example
Once the app is running, navigate to http://localhost:3000 in your browser.
